# Assignment

## Title
How the Internet Delivers a Web Page (Amazon Website)
## Objective
To understand how the Internet delivers a webpage by studying the complete communication process that occurs when a user accesses the Amazon website. This includes understanding the TCP/IP protocol stack, DNS, HTTP, TCP, client–server architecture, cookies, and the communication flow between the browser and the web server.
## Solution
The assignment explains the complete process of loading the Amazon website. It covers:

TCP/IP protocol stack and the role of each layer.
DNS conversion of the domain name into an IP address.
HTTP/HTTPS request and response process.
TCP reliable communication using the Three-Way Handshake.
Client–Server architecture used by Amazon.
Comparison between HTTP and FTP.
Role of cookies with a real-world Amazon example.
Flow diagram showing communication between the browser and the Amazon web server.

Supporting diagrams are included to clearly illustrate the TCP/IP protocol stack and the complete communication process.


## Output
The study demonstrates that when a user enters www.amazon.com in a web browser, DNS translates the domain name into an IP address, TCP establishes a reliable connection, and HTTP/HTTPS transfers webpage data between the browser and the Amazon web server. The browser then renders the received HTML, CSS, JavaScript, and images to display the Amazon webpage. Cookies improve the user experience by remembering login details, shopping cart items, and user preferences.
## Conclusion
This assignment helped in understanding how different Internet protocols work together to deliver a webpage efficiently. DNS converts website names into IP addresses, TCP provides reliable communication, and HTTP/HTTPS enables data exchange between the browser and the web server. Amazon uses a Client–Server architecture to serve millions of users securely and efficiently. Cookies further enhance the browsing experience by storing user preferences and session information. Overall, application-layer protocols play a vital role in making web browsing fast, reliable, and user-friendly, enabling people to access online services conveniently in their daily lives.
